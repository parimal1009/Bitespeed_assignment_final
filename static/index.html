üîó Linking Logic

Contacts are linked if they share:

  • Same email address
  • Same phone number
  • Oldest contact becomes primary
  • Newer contacts become secondary

üîÑ Contact Creation Rules

  • No existing match ‚Üí Create primary
  • Partial match ‚Üí Create secondary
  • Multiple primaries ‚Üí Convert newer to secondary
  • Exact match ‚Üí Return existing data
